===== help: command list =====
Usage: sparcli [global options] <command> [options] [arguments]

Styled terminal output and interactive prompts for the shell,
powered by the sparcli library.

Global options:
  -h, --help        Show this help and exit
  -V, --version     Show the version and exit
      --no-color    Strip ANSI colors from the output
      --no-markup   Treat [tag] markup as literal text
      --allow-ansi  Pass ANSI escapes in input text through

Output commands:
  print      Print markup-styled text
  panel      Draw a bordered panel around text
  rule       Draw a horizontal rule with optional title
  table      Render CSV/TSV data as a table
  list       Render a bulleted or numbered list
  tree       Render indented lines as a tree
  kv         Render key/value lines as an aligned list
  alert      Show an info/warning/error/success panel
  badge      Print an inline styled badge
  columns    Lay out items side by side in columns

Streaming commands:
  spin       Show a spinner while a command runs
  progress   Draw a progress bar fed by stdin values

Input commands:
  confirm    Ask a yes/no question (exit code = answer)
  input      Prompt for a line of text
  password   Prompt for a masked secret
  number     Prompt for a number
  textarea   Prompt for multi-line text
  select     Choose one or more items from a list
  fuzzy      Fuzzy-search and pick an item
  date       Pick a date from a calendar

Exit codes:
  0  success / confirmed
  1  input cancelled / answered no
  2  error, bad usage or no TTY

Run 'sparcli <command> --help' for the options of a command.
Full reference: docs/cli.md
